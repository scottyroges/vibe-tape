# Text-Based Vibe Input

Skip the seed song picker entirely — go straight to Claude and describe the vibe you want in natural language.

Instead of selecting 3–5 songs, the user types something like "chill Sunday morning" or "driving at night through the desert" and Claude generates the vibe criteria directly from the description. The playlist is then scored against the user's library the same way it would be with seed songs.

Could work as an alternative entry point alongside the seed picker, or as a fallback when the user doesn't know which songs to pick.
