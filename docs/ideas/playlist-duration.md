# Playlist Duration

Let the user specify how long they want the playlist to be in hours (e.g., "I need 3 hours of music for a road trip").

- Use the duration to calculate roughly how many songs to fetch, based on average track length
- Could offer presets: quick listen (30 min), session (1 hr), background (3 hr), all day (6+ hr)
- Longer playlists need more variety to avoid fatigue — might need to loosen the vibe criteria slightly as the playlist grows
- Short playlists can be tighter and more curated

This gives the user practical control over output size without making them think in terms of track counts.
